# Smart-contact-manager



Overview
Smart Contact Manager is a web-based application designed to manage and organize contacts efficiently. Built using Spring and Spring Boot, it provides users with a secure, user-friendly interface to store, update, and manage contact information. The application supports features like user authentication, role-based access, and CRUD operations on contacts.

Features
User Authentication: Secure login and registration with role-based access.
Dashboard: Personalized dashboard displaying user-specific contacts.
CRUD Operations: Create, read, update, and delete contacts.
Responsive Design: Fully responsive design for optimal use on various devices.
Data Validation: Comprehensive input validation for user and contact data.
Exception Handling: Robust error and exception handling mechanisms.
